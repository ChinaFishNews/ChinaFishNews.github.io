常用写作命令

  • 建立新文章：hexo n “新文章名”
  • 预览文章：hexo s
  • 生成网页：hexo g
  • 发布文章：hexo d
  • 生成网页并发布文章：hexo d -g
  • 查看Hexo的版本: hexo version
